# 📒 Note_App

A simple and easy-to-use note-taking application built with **Flutter**.  
You can quickly add, edit, and delete your notes.

## 🚀 Features
- 📌 Add new notes  
- 🗑️ Delete notes  
- ✏️ Edit notes  
- 💾 Save notes locally using Shared Preferences  
- 🎨 Custom app icon
